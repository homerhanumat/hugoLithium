# SublimeR-Hugo

Blogdownify-ing the [**Sublime Hugo Theme**](https://github.com/dt801ts/sublime-hugo-theme).

Features added so far:

* Tags and Categories
* "Aggregator" taxonomy (to send selected posts to, e.g., R-bloggers)
* Option for MathJax
* Additonal social media links
